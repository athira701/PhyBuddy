PhyBuddy

PhyBuddy is a Physics Learning Platform built using the MEAN Stack and Clean Architecture principles.
The aim of PhyBuddy is to provide an interactive platform where students can learn Physics through live tutoring sessions, structured learning materials, quizzes, and progress tracking.

Tech Stack

Frontend
- Angular
- TypeScript
- HTML
- CSS

Backend
- Node.js
- Express.js
- TypeScript

Database
- MongoDB

Architecture

- Clean Architecture
- REST API

Project Status

Currently under development.
